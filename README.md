# Data Cleaning Automation

## Project Overview
This project automates the process of cleaning raw salary data using Python and Pandas.

## Features
- Removes missing values
- Cleans duplicate records
- Saves cleaned data to a new CSV file
- Generates a visualization of the salary data

## Technologies Used
- Python
- Pandas
- Matplotlib

## Files
- Salary_Data.csv
- Cleaned_Salary_Data.csv
- data_cleaning.py
- salary_report.png

## How to Run
1. Install Python.
2. Install the required libraries:
   pip install pandas matplotlib
3. Run:
   python data_cleaning.py

## Output
- Cleaned salary dataset
- Salary report graph
